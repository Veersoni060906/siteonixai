Questions

Got questions?

Do I need coding experience?
None at all. You describe what you want in plain English, and Siteonix builds the entire site for you. If you want to customize further, the code is clean and easy to edit.
How do I find clients?
Local niches work best: barbers, salons, realtors, restaurants, tutors. Generate a free preview for a business that has no site, then pitch it to them. The visual does the selling.
Can I really charge $500+?
Yes — especially for businesses without any web presence. You're not just selling a website, you're selling setup, hosting, edits, and an ongoing relationship. The preview makes closing trivial.
What does the AI actually generate?
A complete, single-file HTML page with embedded CSS and JavaScript. Includes hero, features, testimonials, pricing, FAQ, and footer — all tailored to the business type you describe.
Can I edit the output?
Yes. Switch to the Code tab in the builder to edit the raw HTML directly, then click "Apply" to see changes live in the preview. Or export and edit in any text editor.
Is there a free tier?
Yes — you can start building immediately with no credit card required. Generate previews, export code, and test with real clients before committing to a paid plan.
